### 1. Purpose of This Submission

This package proposes targeted additions to the My First Bitcoin (MPB) curriculum focused on **self-custody security mindset** and practical sovereignty education.

### 2. Proposed Contributions

#### A. Self-Custody Security Mindset Module
A 4–6 page educational module covering:
- Custodial vs Self-Custodial Bitcoin
- Counterparty, Surveillance, and Regulatory Risk
- Bitcoin as Sui Generis Bearer Property (light introduction)
- Developing a security mindset
- Common misconceptions and discussion questions

#### B. QuickStart Guide – Sovereign Self-Custody
Practical step-by-step onboarding using Sparrow + self-hosted node + 2-of-3 air-gapped multisig.

#### C. Condensed Risk Framework Overview
Strategic briefing introducing ISA/IEC 62443 defence-in-depth applied to Bitcoin custody.

### 3. Alignment with MPB

These materials complement MPB’s strengths in Bitcoin history and technical basics by adding:
- Explicit focus on security mindset and institutional vs individual risk
- Connection between philosophical foundations and practical tooling
- Current 2026 regulatory context
